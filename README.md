# 📊 Campaign Performance Dashboard | Power BI

## 📌 Project Overview
This project presents a Campaign Performance Dashboard built using Power BI to analyze business performance during campaign and non-campaign periods.

The dashboard helps understand revenue trends, refunds, and customer activity during marketing campaigns.

---

## 🎯 Objectives
- Compare campaign vs non-campaign performance
- Analyze revenue and refunds
- Measure campaign impact on sales

---

## 📊 Dashboard Features
✔ KPI Indicators  
✔ Campaign vs Non-Campaign Analysis  
✔ Revenue Insights  
✔ Refund Analysis  
✔ Interactive Filters

---

## 🛠 Tools Used
- Power BI
- Data Modeling
- DAX
- Data Visualization
